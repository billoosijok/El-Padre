# El Padre

The public website for El Padre, a premium restaurant and bar. It presents the establishment's offerings (menus, drinks, brunch), takes reservations, and is available in French (default), English, and Spanish.

## Language

### Brunch

**Brunch Landing**:
The single, visual-first page dedicated to El Padre's brunch offering. It sells the experience (ambiance, signature dishes, hours) and contains the brunch menu inline as a section rather than linking out to it.
_Avoid_: Brunch page (ambiguous — could mean the old menu-only page)

**Brunch Menu**:
The list of brunch dishes and drinks (Le Salé, Les Suppléments, Le Sucré, Boissons Spéciales). It is a section within the Brunch Landing, not a standalone page.
_Avoid_: Brunch menu page

**Menu Referral**:
A visitor who arrives via an explicit menu deep link (the Brunch Menu section anchor) and is therefore auto-scrolled to the menu on arrival — e.g. a QR code labelled "carte brunch". Choosing brunch from the Menu Chooser does NOT count: those visitors land on the full Brunch Landing. Separate from the always-present jump-to-menu control every visitor sees.

**Menu Chooser**:
The full-screen split-screen prompt that lets a first-time visitor to the classic menu pick between the brunch offering and the classic (tapas/plats) offering.
_Avoid_: Menu selector, splash
